History

O nascimento da internet. Uma época deslumbrante para se estar vivo.

A World Wide Web foi formalizada em 1990 por Tim Berners-Lee enquanto trabalhava no CERN (Organização Europeia para a Investigação Nuclear).

The birth of the Internet. A remarkable time to be alive.

The World Wide Web was formalized in 1990 by Tim Berners-Lee while working at CERN (the European Organization for Nuclear Research).

Nos primeiros anos, a Web era utilizada principalmente por universidades e centros de pesquisa para compartilhar documentos e resultados científicos. Devido às limitações tecnológicas da época, as páginas eram simples e compostas quase exclusivamente por texto.

During its early years, the Web was used primarily by universities and research centers to share documents and scientific findings. Due to the technological limitations of the time, web pages were simple and composed almost entirely of plain text.

A primeira página da Web ainda pode ser encontrada em:

info.cern.ch/hypertext/WWW/TheProject.html

The first web page can still be found at:

info.cern.ch/hypertext/WWW/TheProject.html

Em 1993, a World Wide Web foi disponibilizada gratuitamente ao público. Pessoas no mundo inteiro, com infraestrutura suficiente, puderam então construir seus próprios domínios dentro da rede.

In 1993, the World Wide Web became freely available to everyone. People around the world with sufficient infrastructure could then create their own domains on the network.

Um dos maiores marcos da internet, no mesmo ano, foi a criação do NCSA Mosaic, na Universidade de Illinois, EUA. Como primeiro navegador amplamente adotado, o programa permitia que imagens pudessem ser incluídas como parte do conteúdo das páginas web.

One of the Internet's greatest milestones that same year was the creation of NCSA Mosaic at the University of Illinois, US. As the first widely adopted graphical web browser, it allowed images to be displayed as part of web page content.

Anos 90: Web 1.0

The 90s: Web 1.0

A internet era limitada. Ainda assim, os criadores de conteúdo dedicavam todo o seu esforço e exploravam ao máximo as ferramentas disponíveis na época para dar vida às suas páginas web.

The internet was limited. Even so, content creators devoted all their effort and made the most of the tools available at the time to bring their web pages to life.

A interação era bastante limitada. A maioria das páginas seguia uma filosofia de "somente leitura", com pouco ou nenhum conteúdo gerado pelos usuários.

Interaction was very limited. Most websites followed a "read-only" philosophy, with little or no user-generated content.

O mercado de navegadores, ainda assim, não era um monopólio. Em 1994, Marc Andreessen, junto a outros funcionários, saem da NCSA e fundam sua própria companhia.

class Animal {
    constructor(voice){
        this.voice = voice || 'grunt';
    }

    speak() {
        display(this.voice)
    }
}

class Cat extends Animal {
    constructor(name, color) {
        super('Meow');
        this.name = name;
        this.color = color;
    }
}

const alunos = [
    { nome: "Carlos", notas: [7, 8, 9] },
    { nome: "Ana", notas: [5, 6, 4] }
];

function calcularMedia(notas) {
    const soma = notas.reduce((total, nota) => {
        return total + nota;
    }, 0);

    return soma / notas.length;
}

alunos.forEach(aluno => {
    const media = calcularMedia(aluno.notas);

    console.log(`\nAluno: ${aluno.nome}`);
    console.log(`Média: ${media.toFixed(2)}`);

    if (media >= 7) console.log("Status: Aprovado");
    else console.log("Status: Reprovado");
});
